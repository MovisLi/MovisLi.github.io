# 70. 爬楼梯

斐波那契数列，一个数等于他上一个数和上上的数的和。分为递归解和非递归解，如果有空间限制可以把列表看作一个栈。

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        lst = [1, 1]
        for i in range(0, n-1):
            lst.append(lst[i]+lst[i+1])
        return lst[n]
```
