# 190. 颠倒二进制位

注意下题目中的32位无符号整数。暴力解：

```python
class Solution:
    def reverseBits(self, n: int) -> int:
        return int(str('0'*(32-len(str(bin(n))[2:]))+str(bin(n))[2:])[::-1], 2)
```
