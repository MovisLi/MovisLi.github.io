# 504. 七进制数

暴力法。

```python
class Solution:
    def convertToBase7(self, num: int) -> str:
        if num > 0:
            neg = False
        elif num < 0:
            neg = True
            num = -num
        else:
            return '0'
        n = 0
        while 7**n <= num:
            n += 1
        n -= 1
        res = ''
        while n>=0:
            i = num//(7**n)
            res += str(i)
            num -= (7**n)*i
            n -= 1
        return f'-{res}' if neg else res
```

然而，将返回结果的字符串换成整数之后，速度提升了很多。这里可以看出字符串连接比整数相加效率要低。

```python
class Solution:
    def convertToBase7(self, num: int) -> str:
        if num >= 0:
            neg = False
        else:
            neg = True
            num = -num
        n = 0
        while 7**n <= num:
            n += 1
        n -= 1
        res = 0
        while n>=0:
            i = num//(7**n)
            res += (10**n)*i
            num -= (7**n)*i
            n -= 1
        return '-'+str(res) if neg else str(res)
```
