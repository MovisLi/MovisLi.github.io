# 387. 字符串中的第一个唯一字符

先想到的一种思路就是用双`set`，一个`set`用来存储字符串每个字符，作用时校验是否重复，另一个`set`用来存储重复的字符。

```python
class Solution:
    def firstUniqChar(self, s: str) -> int:
        hashset1 = set('')
        hashset2 = set('')
        for value in s:
            if value not in hashset1:
                hashset1.add(value)
            else:
                hashset2.add(value)
        for index,value in enumerate(s):
            if value not in hashset2:
                return index
        return -1
```

当然用字典也可以。

```python
class Solution:
    def firstUniqChar(self, s: str) -> int:
        hashmap = {}
        for i in s:
            if i not in hashmap:
                hashmap[i] = 1
            else:
                hashmap[i] += 1
        for index,key_char in enumerate(s):
            if hashmap[key_char] == 1:
                return index
        return -1
```
