# 58. 最后一个单词的长度

反向遍历，用一个`flag`稍微控制下流程就行了，注意几个`if`的顺序。

```python
class Solution:
    def lengthOfLastWord(self, s: str) -> int:
        res = 0
        start = False
        for i in range(len(s)-1, -1, -1):
            if s[i] != ' ' and not start:
                start = True
            if s[i] == ' ' and start:
                break
            if start:
                res += 1
        return res
```
