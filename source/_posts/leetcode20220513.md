# 83. 删除排序链表中的重复元素

基本链表操作。

```python
class Solution:
    def deleteDuplicates(self, head: ListNode) -> ListNode:
        if not head:
            return head
        node = head
        while node.next:
            if node.val == node.next.val:
                node.next = node.next.next
            else:
                node = node.next
        return head
```

# 88. 合并两个有序数组

本题要求`inplace`，但是我们仍然可以用`不inplace`的思想做`inplace`的事情。

```python
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        left = 0
        right = 0
        res = []
        while (left+right)<(m+n):
            if left>=m:
                res.append(nums2[right])
                right += 1
                continue
            if right>=n:
                res.append(nums1[left])
                left += 1
                continue
            if nums1[left] <= nums2[right]:
                res.append(nums1[left])
                left += 1
            else:
                res.append(nums2[right])
                right += 1
        nums1[:]=res
```

当然我们也可以逆向开始放数，空间复杂度会更低。

```python
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        tail = m+n-1
        while m>=1 or n>=1:
            if m<1:
                nums1[tail] = nums2[n-1]
                tail -= 1
                n -= 1
                continue
            if n<1:
                break
            if nums1[m-1] >= nums2[n-1]:
                nums1[tail] = nums1[m-1]
                m -= 1
                tail -= 1
            else:
                nums1[tail] = nums2[n-1]
                n -= 1
                tail -= 1
```

# 94. 二叉树的中序遍历

递归：

```python
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res = []
        if root:
            res.extend(self.inorderTraversal(root.left))
            res.append(root.val)
            res.extend(self.inorderTraversal(root.right))
        return res
```

非递归：

```python
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res = []
        if not root:
            return res
        node = root
        stack = []
        while node or stack:
            if node:
                stack.append(node)
                node = node.left
            else:
                node = stack.pop()
                res.append(node.val)
                node = node.right
        return res
```

# 118. 杨辉三角

就按照杨辉三角的样子，从第三行开始，除了左右两个1，其他数等于上一行的邻近两个数的和。

```python
class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        if numRows == 1:
            return [[1]]
        res = [[1],[1,1]]
        curRow = 2
        while curRow<numRows:
            temp = [1]
            for i in range(len(res[curRow-1])-1):
                temp.append(res[curRow-1][i]+res[curRow-1][i+1])
            temp.append(1)
            res.append(temp)
            curRow += 1
        return res
```

# 119. 杨辉三角 II

```python
class Solution:
    def getRow(self, rowIndex: int) -> List[int]:
        res = [1]
        cal_num = (rowIndex)//2
        for i in range(1, cal_num+1, 1):
            res.append(int(res[i-1]*(rowIndex-i+1)/i))
        if rowIndex%2 != 0:
            for i in range(cal_num,-1,-1):
                res.append(res[i])
        else:
            for i in range(cal_num-1,-1,-1):
                res.append(res[i])
        return res
```
