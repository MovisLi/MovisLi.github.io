# 278. 第一个错误的版本

二分查找可以解决，左指针`left`记录当前正确的版本，右指针`right`记录当前错误的版本，所以当`right-left=1`时，`right`所指的就是第一个错误的版本，也就是结果。

```python
class Solution:
    def firstBadVersion(self, n):
        left = 0
        right = n
        mid = (1+n)//2
        while right-left>1:
            if isBadVersion(mid):
                right = mid
            else:
                left = mid
            mid = (left+right)//2
        return right
```
