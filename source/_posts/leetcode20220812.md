# 506. 相对名次

先按逆序排序，排序后用 `dict` 建立分数与名次的对应关系，最后按原分数顺序将名次返回为一个列表。

```python
class Solution:
    def findRelativeRanks(self, score: List[int]) -> List[str]:
        rank_lst = sorted(score, reverse=True)
        rank_dict = {}
        for i in range(len(rank_lst)):
            if i==0:
                rank_dict[rank_lst[i]] = 'Gold Medal'
            elif i == 1:
                rank_dict[rank_lst[i]] = 'Silver Medal'
            elif i == 2:
                rank_dict[rank_lst[i]] = 'Bronze Medal'
            else:
                rank_dict[rank_lst[i]] = str(i+1)
        return [rank_dict[i] for i in score]
```

# 509. 斐波那契数

先来个经典的递归算法。

```python
class Solution:
    def fib(self, n: int) -> int:
        def fibonacci(n):
            if n == 0:
                return 0
            if n == 1:
                return 1
            return fibonacci(n-1)+fibonacci(n-2)
        return fibonacci(n)
```

有点慢。因此我们把中间的值放进一个 `list` ，以空间换时间。

```python
class Solution:
    def fib(self, n: int) -> int:
        lst = [0, 1]
        i = 2
        while i<=n:
            lst.append(lst[i-1]+lst[i-2])
            i += 1
        return lst[n]
```

# 520. 检测大写字母

把这个问题转换为一个状态转换问题，如果当前状态不能兼容上个状态，直接报错。

```python
class Solution:
    def detectCapitalUse(self, word: str) -> bool:
        mode = 0
        for i in word:
            if mode == 0:
                if 97<=ord(i)<=122:
                    mode = 2
                else:
                    mode = 4
            elif mode == 4:
                if 97<=ord(i)<=122:
                    mode = 3
                else:
                    mode = 1
            if mode == 1:
                if ord(i) > 90:
                    return False
            if mode == 2:
                if ord(i) < 97:
                    return False
            if mode == 3:
                if ord(i) < 97:
                    return False
        return True
```
