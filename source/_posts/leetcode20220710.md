# 401. 二进制手表

枚举法，枚举时和分，如果满足就添加进列表。

```python
class Solution:
    def readBinaryWatch(self, turnedOn: int) -> List[str]:
        res = []
        for h in range(12):
            for m in range(60):
               if bin(h).count('1')+bin(m).count('1') == turnedOn:
                   res.append('%d:%.2d' % (h,m))
        return res
```
